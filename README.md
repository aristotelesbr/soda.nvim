# Soda.nvim

Soda.nvim is a colorscheme for Neovim, inspired by the [soda.vim](https://https://github.com/DAddYE/soda.vim) and build with LUA. ðŸŒš

## Installation

### Using Package Manager

```lua
use {
  'aristotelesbr/soda',
  config = function()
    require('soda').setup()
  end
}
```

### Using Vim-Plug

```vim
Plug 'aristotelesbr/soda'

lua << EOF
  require('soda').setup()
EOF
```
